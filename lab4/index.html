Challenge

In this lab, we learned how to create pseudocode. We first pseudocoded an everyday task (Baking Bread) and then moved to pseudocoding a computer task (Simple Calculator).

Problems

The only problems we had were deciding how detailed to go with each part of process for the everyday task, and we ended up deciding on being meticulous in some areas that we agreed needed more details.
